How many workflows come with the vCloud Plug-in?

February 22, 2011

There are a total of 91 new workflows delivered as part of the VMware vCenter Orchestrator Plug-in for vCloud Director. The table below provides their names and descriptions

Workflow NameDescription
Unregister VIM Server Unregisters a vCenter Server.
Import VM as vApp Imports a virtual machine from an available vCenter Server as a vApp.
Power Off VM Powers off a virtual machine. Does not free the resources reserved for the virtual machine.
Publish Catalog Publishes or unpublishes a catalog to all organizations external to the catalog's organization.
Run a full provisioning cycle Configures a new vCloud Director instance. The workflow performs the following actions: * Attaches an existing vCenter Server to the vCloud Director host. * Creates a new provider vDC backed by an existing resource pool from the vCenter Server. * Prepares all ESX hosts from the provider vDC. * Creates a new provider network backed by an existing network from the vCenter Server. * Adds a new organization. * Adds a new organization vDC to the organization backed by the provider vDC. * Adds a new bridged organization network to the organization backed by the provider network. * Adds a new catalog to the organization. * Uploads a new media and adds it to the catalog.
Eject Media Ejects media from a virtual machine.
Shutdown VM Sends a notification to the virtual machine guest to shut down.
Shutdown vApp Sends a notification to the vApp's guest virtual machines to shut down.
Upload Media Uploads a media file. The uploaded media can be an ISO file or a floppy file.
Reconnect VIM Server Tries to force a reconnection to a vCenter Server from its vCloud Director host.
Delete Catalog Deletes a catalog.
Discard Suspended State vApp Discards the state of a suspended vApp.
Update Organization Updates an organization.
Reset VM Resets a virtual machine.
Undeploy VM Stops or suspends a virtual machine and frees the resources reserved for the virtual machine.
Add Connection Adds a new vCloud Director host to the plug-in configuration.
Delete User Deletes a user.
Update vApp Updates a vApp.
Capture vApp Captures a vApp as a vApp template.
Provision new network pool backed by a new DVS port group Creates a new vCloud network pool backed by newly created vSphere port groups. The workflow creates a number of port groups with the specified number of ports inside first, and then creates a network pool backed by those port groups. The created port groups are backed by an existing Distributed Virtual Switch from a cluster inside one of the vCenter Server hosts attached to the vCloud Director host. A network pool is a collection of virtual machine networks that are available to be consumed by vDCs to create vApp networks and by organizations to create organization networks. Network traffic on each network in a pool is isolated at layer 2 from all other networks.
Update Catalog Item Updates a catalog item.
Deploy vApp Deploys a vApp and optionally powers it on.
Add VMW Provider vDC Adds a provider vDC to a vCloud Director host.
Compose vApp Composes a vApp from virtual machines of other vApps.
Add User Adds a user to an organization or imports it from LDAP.
Get Thumbnail Gets a screenshot of a virtual machine's desktop.
Delete Network Pool Deletes a network pool.
Update vDC Updates a vDC.
Update Group Updates a group.
Deploy VM Deploys a virtual machine and optionally powers it on.
Delete VMW Provider Network Deletes an external network.
Clone vApp Clones a vApp.
Add VMW Provider Network Adds an external network to a vCloud Director host.
Update vApp Template Updates a vApp template.
Share Catalog Updates the sharing configuration of a catalog.
Suspend vApp Suspends a vApp. Does not free the resources reserved for the vApp.
Upgrade VMW Host Tries to upgrade an ESX host.
Download vApp Template Downloads a vApp template as an OVF.
Delete Role Deletes a role.
Add Organization Adds an organization to a vCloud Director host.
Reboot VM Sends a notification to the virtual machine guest to reboot.
Update User Updates a user.
Add Network Pool Adds a network pool to a vCloud Director host.
Update VMW Provider vDC Updates a provider vDC.
Add Catalog Adds a catalog to an organization.
Provision provider vDC on top of new ESX server Attaches a new ESX host to the vCloud Director host and optionally creates a new provider vDC backed by the attached ESX host. The ESX host is attached to one vCenter Server host inside the vCloud and it is included in one cluster inside the vCenter Server host. If the cluster does not exist, it will be created. An existing NFS shared datastore is attached to the ESX host. After the ESX host is ready, a new provider vDC can be created inside the vCloud, and it will be automatically backed by the new ESX host. A provider vDC is a group of compute, memory, and storage resources from one vCenter Server host. You can allocate portions of a provider vDC to your organizations using vCloud Director.
Share vApp Updates the sharing configuration of a vApp.
Update Catalog Updates a catalog.
Provision new Media new vApp Template and new vApp Provisions a given organization with a new media, a new vApp template, and a new vApp. Both the media and the vApp template are uploaded within the workflow and the vApp is instantiated from the vApp template with default parameters. Using the default parameters, the instantiated vApp is not deployed, not powered on, with accepted EULA, and with no connected networks.
Update Role Updates a role.
Delete Connection Deletes a vCloud Director host from the plug-in configuration.
Delete Media Deletes a media file.
Print Licensing Reports Prints the licensing reports of a vCloud Director host.
Provision new Organization with default settings Creates a new organization with the following default settings: * It is enabled and allows publishing catalogs to all organizations. * It does not use LDAP. * It uses the default SMTP and notification settings from the vCloud Director host. * It has unlimited Running VM quota and Stored VM quota. * It contains one vDC and two catalogs (one public and one private). * For the vDC the default settings are applied when required: Percentage of CPU guaranteed is 0%, Percentage of Memory guaranteed is 100%, Unlimited VM Quota is set to 'false', Maximum number of VMs is 100 and Maximum number of provisioned networks is 1024. You can modify all settings after the organization is created. An organization is the fundamental vCloud Director grouping. An organization contains users, the vApps they create and the resources the vApps use. An organization can be a department in your own company or an external customer you are providing Cloud resources to.
Insert Media Inserts media into a virtual machine.
Delete vDC Deletes a vDC.
Power Off vApp Powers off a vApp. Does not free the resources reserved for the vApp.
Import VM as vApp Template Imports a virtual machine from an available vCenter Server as a vApp template.
Delete VM Deletes a virtual machine.
Add Org Network Adds an organization network to an organization.
Delete Group Deletes a group.
Import Group Imports a group from the configured directory service to an organization.
Reboot vApp Sends a notification to the vApp's guest virtual machines to reboot.
Update VMW Provider Network Updates an external network.
Reset vApp Resets a vApp.
Delete vApp Template Deletes a vApp template.
Wait For Task Waits for a task to be completed.
Power On VM Powers on a virtual machine.
Power On vApp Powers on a vApp.
Delete Org Network Deletes an organization network.
Delete VMW Provider vDC Deletes a provider vDC.
Add Catalog Item Adds a new item to a catalog. The new item can be a media file or a vApp template.
Upload vApp Template Uploads an OVF as a vApp template.
Register VIM Server Registers a vCenter Server to a vCloud Director host.
Delete Catalog Item Deletes a catalog item from a catalog.
Suspend VM Suspends a virtual machine. Does not free the resources reserved for the virtual machine.
Delete Organization Deletes an organization.
Instantiate vApp Template Instantiates a vApp template as a vApp.
Clone Media Clones a media file.
Repair VMW Host Tries to repair an ESX host.
Update Network Pool Updates a network pool.
Acquire Ticket Acquires a ticket for the remote console of a virtual machine.
Delete vApp Deletes a vApp.
Discard Suspended State VM Discards the state of a suspended virtual machine.
Enable VMW Host Enables or disables an ESX host.
Prepare VMW Host Prepares or unprepares an ESX host.
Add Role Adds a role to a vCloud Director host.
Undeploy vApp Stops or suspends a vApp and frees the resources reserved for the vApp.
Update Media Updates a media file.
Add vDC Adds a vDC to an organization.
Update VIM Server Updates a vCenter Server.
